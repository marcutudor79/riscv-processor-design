# RISCV Processor Design Project
This is the repository for the PR1 and PR2 projects of the Execution Platform course, part of Télécom Paris
curriculum. The purpose of it is to design and implement a simple riscv based specification and basic design
of a processor.

# Contents
- [How to setup project](#how-to-setup-for-the-project)
- [Project contents](#project-contents)
- [Who contributed](#who-contributed)

# How to setup for the project
1. Install ```LaTeX Workshop``` extension in VSCode
2. Install LaTeX (Ubuntu)
```
    sudo apt install latexmk
```
3. Build main.tex using ```LaTeX Workshop``` extension
4. Profit!
